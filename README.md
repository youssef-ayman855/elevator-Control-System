# Elevator Control System

## Description
This project implements a microcontroller-based elevator control system
using ATmega32. It allows users to select floors using a keypad or buttons,
controls a DC motor for movement, and displays the current floor on an LCD.

## Features
- Floor selection (1–5)
- LCD display
- Keypad input
- DC motor control (Up / Down)
- Proteus simulation

## Tools & Technologies
- ATmega32
- MikroC
- Proteus
- Embedded C

## Project Structure
- Code/      → Embedded C source code
- Proteus/   → Proteus simulation files
- Report/    → Project report (PDF / Word)
